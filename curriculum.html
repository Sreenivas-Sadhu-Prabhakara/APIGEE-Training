Instructor flow

Day-by-day Apigee training plan

The program is three calendar weeks. Weekdays are instructor-led build days, Saturdays are clinic and challenge days, and Sundays are consolidation days with reading, refactoring, and review. Every day has explicit practice work so learners leave with muscle memory, not just notes.

Architecture spine

Three-tier API platform model

The course keeps returning to the same layered model: experience clients, Apigee/API management, and backend systems. Learners learn where each decision belongs and what must never be pushed into the wrong tier.

Tier 1Experience and channel clients

Mobile, web, partner apps, internal tools, and developer portal consumers.

Tier 2Apigee X API management

Proxy routing, policies, products, security tiers, mediation, analytics, and shared flows.

Tier 3Backend services and data

Cloud Run, GKE, legacy services, identity systems, consent stores, ledgers, and observability sinks.
